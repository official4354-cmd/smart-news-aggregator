About Smart News Aggregator

The Smart News Aggregator Using NLP is an intelligent web application that automatically collects real-time news, summarizes lengthy articles, and organizes them into categories like Technology, Sports, and Politics. It is developed using Flask, Python, and Transformer-based NLP models.

This project demonstrates how Natural Language Processing can help users save time and access information efficiently in today's fast-moving digital world.
